Cleans up the Valheim main menu by removing visual clutter and mod-related UI elements.

## Features
• Hide "modded" warning text  
• Hide showlog button  
• Hide changelog panel  
• Hide merch store  

All features are configurable via BepInEx config.

## Config
After first launch, edit:
BepInEx/config/crevitka.cleanmenu.cfg

You can enable/disable each element individually.

## Compatibility
Client-side only.  
Safe to use with any modpack.
